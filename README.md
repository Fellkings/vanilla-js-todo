# Vanilla JS Todo List

## Описание проекта
Легковесное веб-приложение для управления задачами (Todo List). Проект реализован с использованием чистых функций (Reducer) и покрыт тестами. Поддерживает добавление, удаление, изменение статуса задач и фильтрацию, а данные сохраняются в куки браузера.

## Стек технологий
* Frontend: HTML5, CSS3, JavaScript
* Тестирование: Tape
* Хранилище: LocalStorage

## Запуск проекта локально
1. Склонируйте репозиторий: `git clone https://github.com/Fellkings/vanilla-js-todo`
2. Откройте файл `index.html` в любом современном браузере.
3. Для запуска тестов выполните:
   - `npm install`
   - `npm test` (или `node test.js`)
